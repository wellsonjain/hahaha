Website Project 01

22 October 2016 on html, front end

The struggle you’re in today is developing the strength you need for tomorrow.

最近在 Hahow 上面看到一門相當超值的課程，叫做 動畫互動網頁程式入門，雖然說是“入門"，可是教材內容卻非常的扎實，二話不說就直接買下來了；在課程內容裡面有一些 Project 的練習，覺得這些也拿來記錄一下也很不錯。

覺得自己還是 Foolstack developer，距離 Fullstack developer 還有好長一段路啊！前端一直是我蠻苦手的地方，就像這個 project 練習我大概一個月前就做過了，但是過了一個月後我竟然還是忘記自己當初到底怎麼讓他完成的，甚至其中設計的概念也不太記得了，所以就像上一篇說的，我還是需要一些軌跡的紀錄讓未來的自己可以有效的還原當初的想法，下面的例子是很單純的一個 HTML 框架，要做的是一張線上的名片，這邊有個很棒的習慣就是以 border 來看出整體的架構是如何安排。

* {
  border: 1px solid black;
}

See the Pen Project 1 by Tzu Chien Wei (@wellsonjain) on CodePen.

因為還沒有做到 Responsive 的關係，最後的結果在 codepen 的 embed 會有超出範圍的狀況發生。這邊 CSS 的部分有幾個地方蠻值得一提的，我們從外層講到內層，首先是 body 的部分，從上面的預覽可以看到，在 html 和 body 的地方都會有一些間隔，我們希望是能夠完全填滿整個頁面，所以在實作上我們會把 padding 跟 margin 給取消，並且給予 100% 的寬高，

html, body {
  width: 100%; height: 100%;
  padding: 0px; margin: 0px;
}

再來就是我們在 body 的地方給了上下兩條粗線，在 html 裡面的 box model 中，border 也是佔寬度的，所以線給的越粗，就會有把內容給擠下去的感覺，這時候就可以使用 border-box 讓它的作用範圍會是在 border 之內

body {
  background-color: #d1d1d1;
  border-top: solid 20px #41B883;
  border-bottom: solid 20px #41B883;

  box-sizing: border-box;
}

在 namecard 的主體內可以藉由 box-shadow 來對整體的區塊去做陰影，在顏色的參數最好是可以使用到透明度，這樣可以有疊上去的感覺

.namecard {
  padding: 20px;
  width: 350px;
  margin: 100px auto 0 auto;
  letter-spacing: 1px;
  color: #666;
  background-color: #eee;

  overflow: hidden;

  border-radius: 5px;
  border: solid 1px rgba(0, 0 , 0, 0.15);

  font-weight: 500;
  box-shadow: 0px 0px 20px rgba(0, 0, 0, 0.2);
}

最後就是定位，在名片中有兩個圓形分別在不同的位置，這邊是用 position: absolute 去實現出來的，absolute 的特性是不參與排序，但是他會相對於上層容器做偏移，而上層容器如果不是可被定位的元素(預設的 staic 是不可被定位)，他就會去對整個 body 做定位偏移，所以在前面的 wildcard selector 就可以去設定所有的 position 都預設為 relative，這樣就可以確保所有都可被定位。

.circle {
  width: 100px;
  height: 100px;
  border-radius: 100%;
  position: absolute;
}
