Basic Javascript Object

25 March 2016 on javascript

You got this. Make it happen. - Danielle LaPorte

What is Object? Object is a “container” of multiple related data, call properties. Like a Book may have the properties, which is title, author, publisher, numChapters, numPages, illustrator.

In javascript, the properties can refer to numbers, strings, arrays, functions, and even other objects!

var myBox = { };
// A set of curly brackets says to make a new object...in this case, however, it’s an empty one with no properties.

myBox = {
    height: 6,
    width: 8,
    length: 10,
    volume: 480,
    material: "cardboard",
    contents: ["Great Expectations", "The Remains of the Day", "Peter Pan"]
};

We can also initialize the content property through using array variable.

var booksArray = ["Great Expectations", "The Remains of the Day", "Peter Pan"];
var myBox = {
    height: 6,
    width: 8,
    length: 10,
    volume: 480,
    material: "cardboard",
    contents: booksArray
};

And the dot operator allows modification of properties, even using methods. After an object has been created, properties can continue to be added. Besides, we can also use “String” as a key, but it can’t use dot operator

 myBox.volume = myBox.length * myBox.width * myBox.height;

 // Since we’re only referring to booksArray, pushing to myBox.contents (or using any Array method) will just modify booksArray!
 myBox.contents.push("On The Road");

myBox.destination1 = "Orlando";
myBox.destination2 = "Miami";

myBox["# of stops"] = 0;

//  myBox

height: 6
width: 12
length: 10
volume: 720
material: "cardboard"
contents: booksArray
weight: 24
destination1: "Orlando"
destination2: "Miami"

Now, we are changing our contents to individual objects. We’ll build a function that creates Book objects and adds them to our Box

First, we’ll delete our contents property with the delete keyword.

delete myBox.contents;
// -> true
// We’ve only deleted the property name and the reference, but not the original booksArray outside the Box.
delete myBox.nonexistentProperty;
// -> true
// delete will return true each time, regardless of whether the property existed or not!

function addBook (box, name, writer){
  box["# of Books"]++;
  box["book" + box["# of Books"]] = {title: name, author: writer};
}

console.log( myBox.book1.title );

An Aquarium Object filled with other objects, first build some add/remove functionality for creatures and environment toys

var aquarium = {
    Nemo: { type: "fish", species: "clownfish", length: 3.7 },
    Marlin: { type: "fish", species: "clownfish", length: 4.1 },
    Dory: { type: "fish", species: "blue tang", length: 6.2 },
    Peach: { type: "echinoderm", species: "starfish", length: 5.3 },
    "Coral Castle": { type: "environment", material: "coquina", moves: false },
    "Dragon Statue": { type: "environment", material: "plastic", moves: false }
};

function addCritter( container, name, type, species, length ){
  container[name] = {type: type, species: species, length: length};
}

function addToy( container, name, type, material, moves ){
  container[name] = {type: type, material: material, moves: moves};
}

Properties can also be FUNCTIONs, and an object’s function properties are often called its “methods”.

var aquarium = {
    Nemo: { type: "fish", species: "clownfish", length: 3.7 },
    Marlin: { type: "fish", species: "clownfish", length: 4.1 },
    Dory: { type: "fish", species: "blue tang", length: 6.2 },
    Peach: { type: "echinoderm", species: "starfish", length: 5.3 },
    "Coral Castle": { type: "environment", material: "coquina", moves: false },
    "Dragon Statue": { type: "environment", material: "plastic", moves: false },

    //  Our container parameter now disappears, since we are making the function BELONG TO that very container.
    addCritter: function (name, type, species, length ){
        // When called with this, addCritter says: Hey, aquarium! Make a new property called name and assign to it a new Object with these properties!
        this[name] = {type: type, species: species, length: length};
    }
};

aquarium.addCritter("Bubbles", "fish", "yellow tang", 5.6);

And then, let’s build another method that removes any object from our aquarium.

aquarium.takeOut = function ( name ) {
    this[name].name = name;
    var temp = this[name];
    delete this[name];
    return temp;
};

var fishOutOfWater = aquarium.takeOut("Marlin");
console.log( fishOutOfWater );
//->  Object {type: "fish", species: "clownfish", length: 4.1, name: "Marlin"}

Now, what if we wanted to know how many fish our tank has at any given time?

aquarium.countFish = function ( ) {
    var numFish = 0;

    // The in keyword looks “in” the Object to its right and finds each enumerable property in turn. Think of it like accessing each index of an Array.
    for(var key in this){
        if ( this[key].type == "fish" ) {
        numFish++; }
    }
    return numFish;
}
