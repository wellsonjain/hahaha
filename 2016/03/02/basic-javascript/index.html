The mind, once stretched by a new idea, never regains its original dimensions. - Oliver Wendell Holmes The basic function structure

Here is basic function structure:

function sumOfCubes (a, b) {
  // do some stuff

  return // something (or nothing) from the process
}

Building function within code execution rather than at program load time

// Build in memory immediately when the program loads
function diffOfSquare(a, b){
  return a*a - b*b;
}

// Function builds ONLY when this line of code is reached
var diff = function diffOfSquare(a, b){
  return a*a - b*b;
};

Since we used the variable name, the function name is optional

// It's called Anonymous functions
diff = function (a, b){
  return a*a - b*b;
};

Besides, the function can be argument, we can also use function as return value. Something like this

// Return function from a function
function hello() {
  ...
  return function itsme()
}

For example, now we want to build a ticket system which have an array as a queue for Fast Pass delivery


var parkRides = [ ["Birch Bumpers", 40] , ["Pines Plunge", 55] , ["Cedar Coaster", 20] , ["Ferris Wheel of Firs", 90]  ];
var fastPassQueue = [ "Cedar Coaster", "Pines Plunge", "Birch Bumpers", "Pines Plunge" ];

In array, we can use push() to add cell to the back of array, and use shift() will return the cell that it removes from the front of the array.

And here is our ticket function,

function buildTicket (allRides, passRides, pick){
  if (passRides[0] == pick) {
    var pass = allRides.shift()
    return function () { alert("Quick! You've got a Fast Pass to" + pass + "!") }
  } else {
    for(var i = 0; i<allRides.length; i++){
      if(allRides[i][0] == pick){
        return function () { alert("A ticket is printing for " + pick + "!\n" + "Your wait time is about " + allRides[i][1] + " minutes."); };
      }
    }
  }
}

var wantsRide = "Birch Bumpers";
var ticket = buildTicket( parkRides, fastPassQueue, wantsRide );
ticket(); // To call the function contained in the ticket variable, we need a set of parentheses and a semicolon.

We will get the message A ticket is printing for Birch Bumpers! Your wait time is about 40 minutes.

But what if we remove the ticket and just call the buildTicket function? Turns out it will return the whole function expression. We need more to call if we want to use an immediately-invoked function

buildTicket( parkRides, fastPassQueue, wantsRide )();
// We give parameter parentheses and a semicolon
//   gives the instruction to execute the function!
